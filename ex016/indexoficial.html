Termo tipografia

Vem de "týpos' 'graphía'

Em resumo é o estudo dos tipos de escrita

Emoções

Assim como as cores, as fontes podem passar sentimentos e emoções, na qual são escolhidas com determinado objetivo

Anatomia dos tipos

Toda fonte quando é desenvolvida, é importate ver os espaços,(vertical, horizontal, etc..., é importante saber essas medidas)

Existe a altura-x, altura ascendete, altura das maiúsculas, altura descendente e altura do corpo (altura total da letra)

Lembrar

-> div especial, indica que o conteúdo é principal
width -> largura da caixa
padding -> espaço lateral do texto dentro da caixa
margin -> margem do texto (auto, 3px, ...)
text-align -> alinhamento do texto (center, end, justify, ...)
border-radius -> cantos da caixa arredondados
box-shadow / text-shadow -> sombra da caixa
- parâmetros: tam. sombra p/ lado, tam. sombra p/ baixo, espalhamento, cor;
Ex: box-shadow: 5px 5px 15px rgba(138, 111, 95, 0.616);
text-shadow: 1px 1px 2px rgba(129, 78, 5, 0.733);
